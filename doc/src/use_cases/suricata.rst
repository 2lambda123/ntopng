Suricata
########

ntopng integrates with the Intrusion Detection System (IDS) Suricata to augment the traffic with security metadata (see :ref:`Suricata Integration`). Blog post `ntopng & Suricata: Unifying Visibility with Security <https://www.ntop.org/ntopng/ntopng-suricata-unifying-visibility-with-security/>`_ explains this in detail.
